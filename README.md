# Hospital-ER-Analysis-Excel
End-to-end data analysis project using Advanced Excel, Pivot Tables, and Interactive Dashboards.

## 📌 Project Overview

This project focuses on analyzing patient flow within a Hospital Emergency Room (ER). The primary objective is to monitor patient volume, understand wait time dynamics, and provide data-driven insights to improve operational efficiency and patient care.

## 📊 Key Insights & Metrics

* **Total Patient Volume:** Analyzed overall traffic to identify trends and load.
* **Wait Time Analysis:** Calculated average waiting times to highlight potential bottlenecks.
* **Admission vs. Discharge:** Tracked the percentage of patients requiring hospital admission versus those discharged.
* **Peak Hour Identification:** Discovered the busiest times of the day to help optimize medical staffing.

## 🛠️ Tools Used

* **Advanced Excel:** Used for data cleaning, transformation, and complex calculations.
* **Pivot Tables & Charts:** Leveraged to summarize large datasets efficiently.
* **Interactive Dashboard:** Created a dynamic visual interface using Slicers and Timelines for real-time data exploration.
